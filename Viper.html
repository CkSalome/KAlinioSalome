Tricks

1. Snake bite Line ups Using a poison orb, you can force the defuser to only have 1hp. Then with a lined up snake bite, if landed on the correct spot, can deal the finishing blow to the defuser. The molly has same trajectory as a brimstone molly so learning brim line ups might be similar.
2. Using a Shotgun in Viper's Pit Using a shotgun when in viper's ultimate is an easy to way to finish off enemies. This is because Viper's pit decays the enemies to 1hp. Shotguns are preffered here because it shoots several bullets at the time making it harder to miss.
